# Інструкція для користувача
## 1. Запуск та підключення
•	Запуск системи: Виконайте команду в терміналі: 
java -jar DigiUni_Registry-1.0-SNAPSHOT-jar-with-dependencies.jar
•	З'єднання: Після запуску встановіть зв'язок із TCP-клієнтом.
## 2. Авторизація
•	Перейдіть у розділ для керування акаунтом.
•	Увійдіть у систему під своїми даними або пройдіть швидку реєстрацію.
•	Важливо: Якщо ви викладач або відповідальна особа, зверніться до адміністратора для отримання відповідних прав доступу.
## 3. Навігація та керування
•	Використовуйте головне меню для перемикання між розділами системи.
•	Для адміністраторів: Спеціальні функції доступні в меню: Account -> Admin Panel.
## 4. Завершення роботи
Щоб коректно завершити сесію, натисніть 0 (Exit) на головній сторінці додатка.
